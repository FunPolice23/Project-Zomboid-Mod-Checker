# PZ Mod Compatibility Checker – Build 42 Edition

Fast, accurate checker for Project Zomboid mods against Build 42.

**Features**
- Full Java + Lua analysis (fragile calls, deprecated hooks, missing methods)
- Color-coded Results tab + 30 beautiful themes
- Workshop scanner with caching
- GUI + CLI support

**Installation**
1. Download the .exe (or run from source)
2. All files (caches, reports, crash.log) saved to `Documents/PZModChecker`

**Usage**
- GUI: Run the .exe, select mod, click Run
- CLI: `PZModChecker.exe game.jar mod_folder --lua-only`

**Notes**
- Works on Windows (Steam auto-detect). Linux/Mac users can browse manually.
- Report saved automatically on "Save Report".
